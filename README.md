# seaborn_analyzer
seabornを利用して、
以下のパッケージからなります
|パッケージ名|概要|
|---|---|
|custom_pair_plot.py|散布図行列と相関係数行列を同時に表示|
|custom_hist_plot.py|ヒストグラムと各種分布のフィッティング|
|custom_scatter_plot.py|相関分析、回帰および分類結果の散布図表示|

## custom_pair_plot.py
散布図行列と相関係数行列を同時に表示するライブラリです

## custom_hist_plot.py
